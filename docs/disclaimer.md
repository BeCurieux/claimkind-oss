# Disclaimer

**ClaimKind performs deterministic linting against citation-backed rules. It is
not legal advice.**

Findings, citations, and remediation guidance are provided to inform review.
Compliance decisions remain with the brand and its counsel. Position ClaimKind
exactly like a security scanner: findings + citations + guidance, with decisions
staying with the owner. The tool points at the regulator's own enforcement
actions; it does not play one.

This block appears in the README, the docs, and `LintResult`/API response
documentation by design.
